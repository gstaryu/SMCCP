# README

本项目实现了一个**简易多机协同计算平台**，项目中每个文件及功能如下：

```
.
├── control.py          # 控制节点/主节点程序
├── client.py           # 计算节点/工作节点程序
├── generate_data.py    # 用于生成测试数据的脚本
│
├── task1.py            # 任务1的实现代码
├── task2.py            # 任务2的实现代码
└── task3.py            # 任务3的实现代码
│
├── setup1.txt          # 任务1的配置文件
├── setup2.txt          # 任务2的配置文件
└── setup3.txt          # 任务3的配置文件
│
├── run_task.sh         # [Linux/macOS] 运行任务的Shell脚本
└── run_task.bat        # [Windows] 运行任务的批处理脚本
```
